<threat_model>

## Trust Boundaries

| Boundary                                     | Description                                                                                                       |
| -------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| browser URL search params → typed list state | Untrusted query-string values enter the app here and must be normalized before driving queries or table controls. |
| typed list state → query keys                | If scope or params are omitted, cache entries may leak across parent contexts.                                    |
| typed list state → table state               | If table state diverges from URL state, users can see one state while data fetches use another.                   |

## STRIDE Threat Register

| Threat ID | Category | Component                                            | Disposition | Mitigation Plan                                                                                                                                 |
| --------- | -------- | ---------------------------------------------------- | ----------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| T-01-05   | T        | `features/directory/shared/directory-list-state.ts`  | mitigate    | Parse and normalize all query-string values into typed defaults so invalid or missing values cannot silently produce inconsistent fetch inputs. |
| T-01-06   | I        | `features/directory/shared/directory-query-keys.ts`  | mitigate    | Include parent scope and normalized params in list/detail key helpers so cached nested resources remain isolated by context.                    |
| T-01-07   | D        | `features/directory/shared/directory-table-state.ts` | mitigate    | Keep manual pagination/sorting adapters aligned with backend-shaped state to avoid unbounded client-side transforms over partial datasets.      |
| T-01-08   | S        | `features/directory/shared/directory-scope.ts`       | mitigate    | Build scope helpers around stable ids/codes only; do not accept mutable display names as parent identifiers.                                    |

</threat_model>
